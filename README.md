# 📄 HTML Forms Collection

Coleção de 4 projetos iniciantes desenvolvidos em **HTML5 e CSS3**, com foco na criação de formulários estruturados e organizados.

Este repositório demonstra a evolução do nível básico ao mais estruturado na construção de interfaces para coleta de dados, aplicando boas práticas de organização e validação.

📌 Projetos ideais para quem está iniciando no desenvolvimento Front-End.

---

## 📋 Funcionalidades

- Formulário acadêmico completo  
- Formulário simples de contato  
- Cadastro com dados pessoais e login  
- Formulário avançado com múltiplos campos  
- Uso de `fieldset` e `legend`  
- Validações com `required` e `pattern`  
- Upload de arquivos  
- Uso de `radio`, `checkbox`, `select` e `textarea`  
- Organização visual com CSS interno  
- Utilização de CSS Grid  

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  

---

## 🧩 Projetos Inclusos

---

### 1️⃣ Formulário Acadêmico

Formulário completo contendo:

- Dados pessoais  
- Dados acadêmicos  
- Seleção de curso (radio)  
- Áreas de interesse (checkbox)  
- Campo para mini currículo  
- Botões de envio e limpeza  

**Conceitos aplicados:**

- Estrutura HTML5  
- `<fieldset>` e `<legend>`  
- `input type="radio"`  
- `input type="checkbox"`  
- `<textarea>`  
- `<select>`  
- Validação com `required`  

---

### 2️⃣ Formulário de Contato

Formulário simples contendo:

- Nome  
- Email  
- Mensagem  

📍 Ideal para treinar estrutura mínima de formulário.

**Conceitos aplicados:**

- Estrutura básica de formulário  
- Validação obrigatória  
- Organização com CSS interno  
- Uso de `textarea`  

---

### 3️⃣ Cadastro Completo com Login

Formulário mais avançado contendo:

- Dados pessoais (CPF, RG, nascimento)  
- Endereço completo  
- CEP com `pattern` para validação  
- Upload de imagem  
- Login e senha com confirmação  

**Conceitos aplicados:**

- `input type="date"`  
- `input type="file"`  
- `input type="password"`  
- Uso de `pattern`  
- Organização por múltiplos `fieldset`  
- Estrutura semântica aprimorada  

📈 Nível intermediário de organização e estruturação.

---

### 4️⃣ Formulário Completo com Grid

Versão mais completa do repositório, incluindo:

- Dados pessoais detalhados  
- Endereço completo  
- Contatos (telefone, celular, fax)  
- Estado civil  
- Sexo (radio)  
- Lista de interesses organizada com CSS Grid  
- Estrutura visual mais organizada  

**Conceitos aplicados:**

- CSS Grid  
- Melhor organização visual  
- Inputs variados  
- Separação clara de seções  
- Estrutura próxima de formulários reais  

🚀 Este projeto demonstra evolução clara na organização do código.
